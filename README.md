
# 🌦️ Live Weather App

A simple and responsive **Weather Application** built using **HTML, CSS, and JavaScript** that fetches real-time weather data using a live API.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays current temperature
* 💨 Shows air quality index (AQI)
* ☁️ Dynamic weather conditions (cloudy, sunny, rainy, etc.)
* ⚡ Fast and real-time API response
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Weather API (e.g., WeatherAPI)

---

## 🔗 API Used

This project uses a free weather API:

* https://www.weatherapi.com/

## ⚙️ How It Works

1. User enters a city name
2. JavaScript captures the input
3. API request is sent using `fetch()`
4. Data is received in JSON format
5. UI updates with temperature and weather details
